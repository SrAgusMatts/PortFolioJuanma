Sobre mí

Soy Juan Manuel Algarbe, desarrollador FullStack con experiencia en el diseño y desarrollo de aplicaciones web modernas, escalables y centradas en el usuario. Me apasiona crear soluciones eficientes que integren tanto el frontend como el backend de forma fluida.

En el frontend, trabajo con tecnologías como HTML, CSS y JavaScript, utilizando frameworks como React para construir interfaces dinámicas y responsivas. Me enfoco en ofrecer experiencias visuales limpias y funcionales.

En el backend, desarrollo APIs RESTful utilizando Node.js y .NET, aplicando buenas prácticas de arquitectura y seguridad. También tengo experiencia en bases de datos relacionales, control de versiones con Git y despliegue continuo mediante Azure DevOps.
